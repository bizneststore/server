Our server features an event engine that provides 2 kind of events. Main (automated) events and mini events.
Main events are automated & random every 1 hour and you can join them using .join/.leave
commands or registering from Tartarus (Event npc) located in Giran.
Current Main events are Team vs Team,Capture the flag
Domination,Mass domination,Last man standing,DeathMatch,Zombies and TvT Advanced
Mini events are not automated and you can join them using the Pillar of Champions which is located
in giran near olympiad monument.Mini events are Single player fights,Party fights,Mini TvT,Korean style
Always remember to use buffs in events,you can configure them from Tartarus or Pillar of Champions.
